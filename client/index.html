Instruction for use:

1. Select a type of working movement (or a group of movements) and name your workout

2. Choose an appropriate working time, the default is set to 2 minutes and 30 seconds.

3. Determine how many repetitions you would like to achieve! This is optional

4. Select an appropriate rest time

5. If you would like to repeat this workout multiple times, select your desired sets.

6. Get that burn on and start your workout!
